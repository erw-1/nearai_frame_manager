# Contributing

Thanks for helping improve the NearAI frame manager! PRs and forks are welcome.

## Quick Guidelines

- Keep behavior changes small and well described.
- Preserve the output naming and folder structure unless discussed.
- Add or update docs when you change CLI behavior.
- Run a small sample dataset before submitting a PR.

## Development

```
pip install piexif
python run_nearai_frame_manager.py --help
```

## Pull Requests

1. Open an issue or explain the change in the PR description.
2. Include a short test note (what you ran, and with which inputs).
3. Keep formatting consistent with the existing codebase.
